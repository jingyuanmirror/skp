## 1. 功能概述

会员入会功能让用户通过一句自然语言即可完成SKP会员注册，无需填写任何表单。Agent识别入会意图后，即时完成"注册"并弹出虚拟会员卡卡片，随后主动询问用户偏好品类以建立用户画像，为后续精准推荐奠定基础。

同时，Agent具备**会员引导感知能力**：当用户咨询会员信息或与会员权益相关的服务时（如停车费优惠），Agent会主动介绍会员权益并引导入会。

**核心价值**：
1. **无感入会**：将传统"打开App → 找会员注册 → 填写手机号/姓名/验证码 → 提交"的多步表单流程，压缩为一句对话交互
2. **主动引导**：在用户触及权益相关场景时，自动识别并引导入会，降低转化漏斗流失
3. **画像初始化**：入会后即时收集品类偏好，为个性化推荐提供冷启动数据

## 2. 用户故事

| # | 角色 | 故事 | 验收标准 |
|---|------|------|----------|
| US-1 | 新用户 | 我说"我想入会"，管家帮我完成注册并告诉我已入会 | Agent回复确认文案，弹出会员卡卡片 |
| US-2 | 新用户 | 我用不同说法表达入会意图，管家也能识别 | "办卡"、"注册会员"、"成为会员"等均触发入会流程 |

## 3. 交互流程

### 3.1 入会主流程（含偏好收集）

```
用户：我想入会
  ↓
Agent：[意图识别 → 入会]
  ↓
Agent 回复：「李先生，目前已经帮您办理完入会。
            您后续可以点击会员中心，进行会员信息查看，
            也可以随时问我会员信息。

            另外，为了更好地服务您，您可以把偏好告诉我，
            比如您喜欢的品类或近期关注的品牌，
            这样有相关信息时，我会第一时间为您推荐。」
  ↓
Agent 弹出会员卡卡片
  - 品牌：SKP · Member
  - 等级：◇ 银卡会员
  - 持卡人：李先生
  - 卡号：SKP 2026 **** 88
  - 权益：[积分累计享双倍] [专属停车优惠] [生日礼遇]

快捷回复：[高奢腕表皮具] [美妆护肤] [Hermès] [Chanel]

  ↓  （用户点击品类快捷回复，如"美妆护肤"；或输入品牌名，如"我喜欢Hermès"；或输入商品，如"最近在看丝巾"）
用户：美妆护肤 / 我喜欢Hermès / 最近在看丝巾
  ↓
Agent：[偏好识别 → 记录画像（品类/品牌/商品）]
  ↓
Agent 回复（以"美妆护肤"为例）：
  「好的，已记下您的偏好，包括美妆护肤。后续有相关新品或活动，我会第一时间为您留意。

    如有其他需要，请随时告诉我。」

Agent 回复（以"我喜欢Hermès的丝巾"为例，同时命中品牌+商品）：
  「好的，已记下您的偏好，包括Hermès和丝巾。后续有相关新品或活动，我会第一时间为您留意。

    如有其他需要，请随时告诉我。」

快捷回复：[今日专属优惠] [查询停车状态] [联系专属SA]
```

### 3.2 会员咨询引导流程

```
用户：商场有会员吗 / 怎么办会员
  ↓
Agent：[意图识别 → 会员咨询]
  ↓
Agent 回复：「李先生，SKP提供完善的会员体系，包含银卡、钻石、黑钻三个等级，
            会员可享受积分累计、专属停车优惠、生日礼遇等多项权益。

            目前入会还可享受当日停车费减免，是否需要为您办理入会？」

快捷回复：[我想入会] [了解会员权益] [稍后再说]
```

### 3.3 权益场景引导入会流程

```
用户：我停车了 / 停车费怎么算 / 有什么优惠
  ↓
Agent：[意图识别 → 权益相关查询 + 未入会状态]
  ↓
Agent 回复：「李先生，目前入会即可享受当日停车费减免等多重会员权益。
            是否需要为您办理入会？」

快捷回复：[我想入会] [了解会员权益] [暂不需要]
```

### 3.4 流程时序

```
用户输入 → 意图识别（多层级正则匹配）→ 显示打字动画（1300ms）→ Agent 回复 + 可能附带卡片/快捷回复
```

## 4. 意图识别规则

## 5. 用户画像数据模型

### 5.1 前端数据结构

```typescript
interface UserProfile {
  categories: string[];      // 用户偏好品类列表，如 ["美妆护肤", "高奢腕表"]
  brands: string[];          // 用户偏好品牌列表，如 ["Hermès", "Chanel"]
  items: string[];           // 用户偏好商品列表，如 ["丝巾", "手袋"]
  _justOnboarded?: boolean;  // 刚完成入会标记（用于偏好收集对话窗口）
}
```

### 5.2 状态管理

- `categories`：持久存储用户选择的品类偏好，支持多品类累积，去重
- `brands`：持久存储用户偏好的品牌，支持多品牌累积，去重
- `items`：持久存储用户偏好的商品类型，支持多商品累积，去重
- `_justOnboarded`：临时标记，入会成功时设为 `true`，用户完成一次偏好选择后设为 `false`
- 当前为会话级状态（刷新丢失），M1 阶段将持久化至后端

### 5.3 偏好确认文案逻辑

Agent 会将所有识别到的偏好（品类 + 品牌 + 商品）合并为一条自然语句：

```
仅品类：
  「好的，已记下您的偏好，包括美妆护肤。后续有相关新品或活动，我会第一时间为您留意。」

品类 + 品牌：
  「好的，已记下您的偏好，包括美妆护肤和Hermès。后续有相关新品或活动，我会第一时间为您留意。」

品牌 + 商品（如"我喜欢Hermès的丝巾"）：
  「好的，已记下您的偏好，包括Hermès和丝巾。后续有相关新品或活动，我会第一时间为您留意。」

品类 + 品牌 + 商品：
  「好的，已记下您的偏好，包括美妆护肤、Chanel和丝巾。后续有相关新品或活动，我会第一时间为您留意。」
```

同一维度多个项使用顿号连接，最后一项前加"和"。

## 6. Agent 回复文案

### 6.1 入会成功 + 偏好引导

```
李先生，目前已经帮您办理完入会。您后续可以点击会员中心，进行会员信息查看，也可以随时问我会员信息。

另外，为了更好地服务您，您可以把偏好告诉我，比如您喜欢的品类或近期关注的品牌，这样有相关信息时，我会第一时间为您推荐。
```

快捷回复：`[高奢腕表皮具] [美妆护肤] [先锋设计师品牌] [高端餐饮]`

### 6.2 偏好确认

```
好的，已记下您对{品类列表}的偏好，后续有相关新品或活动，我会第一时间为您留意。

如有其他需要，请随时告诉我。
```

快捷回复：`[今日专属优惠] [查询停车状态] [联系专属SA]`

### 6.3 会员咨询引导

```
李先生，SKP提供完善的会员体系，包含银卡、钻石、黑钻三个等级，会员可享受积分累计、专属停车优惠、生日礼遇等多项权益。

目前入会还可享受当日停车费减免，是否需要为您办理入会？
```

快捷回复：`[我想入会] [了解会员权益] [稍后再说]`

### 6.4 权益场景引导入会

```
李先生，目前入会即可享受当日停车费减免等多重会员权益。是否需要为您办理入会？
```

快捷回复：`[我想入会] [了解会员权益] [暂不需要]`

### 6.5 文案规范

- 称呼使用姓氏尊称（当前硬编码"李先生"，M1 阶段对接CRM后动态获取）
- 语调遵循高奢规范：谦逊、专业、有边界感
- 引导入会时突出**即时利益**（如"当天停车费减免"），降低决策门槛
- 品类偏好追问不超过2个选择项的原则体现在快捷回复中（4个选项已精简为品类维度而非具体品牌）
- 不使用任何低奢用词（禁用"亲"、"亲亲"等，详见主PRD §3.1）

## 8. 会员卡卡片 UI 设计

### 8.1 视觉规格

| 属性 | 值 |
|------|-----|
| 卡片宽度 | max 300px，跟随气泡宽度扩展 |
| 圆角 | 16px |
| 主背景 | 深色渐变：`linear-gradient(135deg, #1A1713 0%, #2A2520 50%, #1A1713 100%)` |
| 金色边框 | `linear-gradient(135deg, #C9A86C 0%, #B8924A 40%, #D4B978 60%, #B8924A 100%)` 1.5px |
| 阴影 | `0 8px 32px rgba(184,146,74,0.18), 0 2px 8px rgba(0,0,0,0.3)` |
| 弹出动画 | 从底部滑入 + 缩放：`opacity: 0→1, y: 20→0, scale: 0.92→1`，0.5s 弹性缓动 |

| 效果 | 实现方式 | 时长 | 说明 |

| 纹理线条 | CSS `repeating-linear-gradient` | 静态 | 模拟卡面水平纹理质感 |


### 8.4 设计要点

- **材质感**：深色背景 + 金色边框 + 微纹纹理 = 模拟高端实体会员卡的触感
- **品牌感**：左上角 "SKP — Member" 用 Cormorant 衬线体标注品牌归属
- **等级徽章**：右上角半透明金色渐变背景圆角 pill，内含等级图标 + 文字
- **持卡人姓名**：Cormorant 22px 大号衬线字体，浅色前背景 (#F5F2ED)，仪式感强
- **卡号**：DM Mono 11px 等宽字体，灰调 (#8C8278)，中间脱敏处理
- **权益标签**：9px 圆角 pill 标签，半透明金色底+金色字，横排 flex-wrap
- **微光效果**：卡片出现后 0.5s 延迟触发一次从左到右的 shimmer 扫光，模拟卡面反光
