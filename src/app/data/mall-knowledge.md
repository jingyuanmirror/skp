# SKP 商场基础知识库（文档版）

> 版本：v1.0  
> 更新时间：2026-06-27  
> 适用对象：前台客服 / AI 助理 / 运营同学

## 1. 服务台在哪里？

- 标准答复：
  - 商场服务台位于 **1层中庭东侧（近1号门）**。
- 可提供服务：
  - 商场问询
  - 失物招领
  - 轮椅借用
  - 会员服务协助
- 建议补充话术：
  - 如需，我可继续为您指引到最近动线。

## 2. 是否提供轮椅服务？

- 标准答复：
  - 商场提供轮椅借用服务。
- 办理方式：
  - 前往 1 层服务台登记联系电话，通常需凭有效证件办理借用。
- 注意事项：
  - 高峰时段若轮椅数量紧张，按到场顺序安排。

## 3. 会员可以享受多久的免费退货？

- 标准答复：
  - 会员可享受 **7天内免费退货**（在商品完好、吊牌与购物凭证齐全且符合品牌规则的前提下）。
- 规则补充：
  - 部分奢侈品、定制商品、特价商品可能不适用。
  - 具体以门店规则及当期政策为准。

## 4. 是否可以免费邮寄？

- 标准答复：
  - 商场支持在指定条件下的免费邮寄服务。
- 通常条件：
  - 满足当日消费门槛，可享同城或指定区域免费寄送。
- 规则补充：
  - 超出配送范围、特殊时段或特殊品类可能产生运费。
  - 具体以门店和当日活动规则为准。

## 5. 推荐统一回复风格（可选）

- 称呼建议：
  - 李先生 / 尊敬的会员
- 语气建议：
  - 简洁、礼貌、明确；先给结论，再补充条件。
- 风险控制：
  - 涉及时效、费用、品牌特例时，增加“以门店当日规则为准”。

## 6. 餐厅信息

### 6.1 新荣记（7F-D01）
- 菜系：台州菜（米其林三星）
- 特色菜品：家烧黄鱼、沙蒜豆面、熔岩蜜汁红薯、花胶黄鱼羹
- 人均：800–1200元
- 建议：需提前预约，高峰等位约30–60分钟
- 备注：商务宴请首选，包间需提前1天预定

### 6.2 大董（6F-D02）
- 菜系：烤鸭 / 创意中餐（米其林一星）
- 特色菜品：酥不腻烤鸭、董氏烧海参、奇妙虾球、红花汁鳘肚
- 人均：600–900元
- 建议：烤鸭建议整只预定，双人份半只亦可
- 备注：有零点与套餐可选，午市套餐性价比较高

### 6.3 L'Atelier de Joël Robuchon（6F-A01）
- 菜系：法餐（米其林二星）
- 特色菜品：鱼子酱龙虾冻、黑松露和牛塔塔、焦糖布丁
- 人均：1500–2500元
- 建议：吧台位可观开放式厨房，适合1–2位用餐
- 备注：晚市建议提前2天预约，着装要求商务休闲

### 6.4 甬府（7F-C01）
- 菜系：宁波菜（米其林一星）
- 特色菜品：冰镇花雕醉虾、宁式鳝丝、红膏炝蟹、苔菜小方烤
- 人均：500–800元
- 建议：海鲜类菜品建议当季时令，可咨询服务员推荐
- 备注：午市套餐性价比较高

### 6.5 翠园（5F-B01）
- 菜系：粤菜
- 特色菜品：脆皮乳猪、菠萝咕噜肉、脆皮烧鹅、杨枝甘露
- 人均：300–500元
- 建议：周末茶市人气高，建议11:00前到店
- 备注：有儿童座椅，适合家庭用餐

### 6.6 鼎泰丰（B1-A01）
- 菜系：台湾小笼包 / 点心
- 特色菜品：小笼包、蟹粉小笼、虾仁烧卖、酸辣汤、芋泥小笼
- 人均：150–250元
- 建议：全天排队，非高峰（14:00–17:00）等位较短
- 备注：可外带生小笼包，30分钟内口感最佳

### 6.7 喜茶（B1-C01）
- 菜系：新式茶饮
- 特色产品：多肉葡萄、生打椰椰芒、烤黑糖波波牛乳
- 人均：25–40元
- 备注：支持小程序提前下单到店取

### 6.8 ％Arabica（1F-E01）
- 菜系：精品咖啡
- 特色产品：西班牙拿铁、抹茶拿铁、冰美式
- 人均：35–55元
- 备注：门店位于1层中庭，提供外带与少量堂食位
